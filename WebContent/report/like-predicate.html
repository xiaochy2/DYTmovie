In the function of searching, we use SQL LIKE predicate %A% where
all strings(movie name) that contains the pattern "A" anywhere. For instance, "Spider Man" and "A Space Odyssey" both satisfy the condition.

While in the function of browsing, we use SQL LIKE predicate 1% where
all the strings(movie name) that starts with "1". For example, "12 Monkeys" would satisfy the condition.
