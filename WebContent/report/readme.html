All *.class is in the directory of /Fablix/build/classes
All *.java is in the directory of /Fablix/src

To compile:
cd /Fablix/src
javac *.java

To install:
Put all the generated class files in the directory of /Fablix/build/classes

To reload:
export Fablix.war in Eclipse and deploy Fablix.war

**Set java/mysql-connector-java-5.1.40-bin.jar as your classpath
CLASSPATH=.:$CLASSPATH:~/mysql-connector-java-5.1.40-bin.jar
export CLASSPATH
